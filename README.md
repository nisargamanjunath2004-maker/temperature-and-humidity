# temperature-and-humidity

The Temperature and Humidity Monitoring System is an IoT-based project used to measure environmental temperature and humidity in real time. In this project, the DHT11 sensor collects temperature and humidity data from the surroundings and sends it to the ESP8266 NodeMCU. The controller processes the data and uploads it to cloud platforms like ThingSpeak through Wi-Fi. This project is useful in weather monitoring, agriculture, healthcare, and smart environments.
Components Used – Short Note
ESP8266 NodeMCU – Main microcontroller used for processing and Wi-Fi communication.
DHT11 – Measures temperature and humidity values.
Breadboard – Used for temporary circuit connections.
Jumper Wires – Used to connect components.
USB Cable – Provides power and programming connection.
ThingSpeak – Cloud platform used for data monitoring and visualization.
